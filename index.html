Architecture

The 4-Node LangGraph Pipeline

Each node is an autonomous AI agent with its own tools, LLM calls, and fallback logic. State flows sequentially through the graph.

01
🔍

Research

Market Intelligence

Tavily Search BeautifulSoup Groq LLM
  • Queries competitor stores, pricing, trends
  • Scrapes 4+ competitor pages live
  • LLM synthesises structured JSON insights
  • Fallback data if scraping is blocked
Output → research_output.json

🎨

Built With

Production-Grade
Tech Stack

LangGraph

StateGraph with 4 sequential nodes, typed AgentState passing, and clean edge definitions. The backbone of the entire agent pipeline.

Orchestration

Groq + LLaMA 3.3

llama-3.3-70b-versatile for all LLM reasoning. JSON mode for structured extraction. Used across all 4 nodes with custom system prompts.

AI Inference

Tavily Search API

Real-time web intelligence — competitor search, trend analysis, and pricing research. Both search() and search_answer() modes used.

Web Intelligence

Shopify Admin API

REST API 2024-01 for live store deployment. Creates collections, products, and pages. Rate-limited to 2 req/sec for production safety.

Deployment

BeautifulSoup4

Live competitor page scraping for authentic market data. HTML parsing and text extraction from real brand websites.

Web Scraping

Python 3.11

TypedDict state management, robust error handling, fallback chains, dotenv config, and clean modular architecture.

This project was created as part of Perfomity Media's AI Agent Developer screening assignment. The goal: build a fully autonomous agent that takes a simple brand brief and produces a complete, live Shopify store without any manual intervention.

Every component — from the LangGraph pipeline architecture to the fallback logic, from the brand strategy prompts to the rate-limited Shopify deployment — was designed to demonstrate production-grade agent engineering.
